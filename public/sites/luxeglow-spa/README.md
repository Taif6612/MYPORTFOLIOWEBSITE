# LuxeGlow Salon & Spa — Premium Beauty & Wellness

A luxury salon and spa landing page built as a single dependency-free HTML file. LuxeGlow delivers an immersive beauty and wellness experience with elegant dark theming, rich visuals, and smooth animations.

## Features

- **Hero Section** — Cinematic full-bleed imagery with staggered entrance animations
- **Services Menu** — Hair, skincare, massage, nail, and spa treatment listings with pricing
- **Treatment Showcase** — Featured services with detailed descriptions and booking CTAs
- **Online Booking** — Appointment form with service, date, time, and stylist selection
- **Gallery** — Salon ambience and treatment result photography
- **Team Profiles** — Stylist and therapist bios with specialties
- **Testimonials** — Client reviews with star ratings
- **Dark / Light Mode** — Luxe dark UI with warm gold accent tones
- **Fully Responsive** — Optimized for all screen sizes

## Tech Stack

- **HTML5** — Semantic, SEO-friendly structure with Open Graph meta tags
- **CSS3** — Custom properties, glassmorphism, scroll-reveal animations, responsive grid
- **Vanilla JavaScript** — Zero dependencies, zero build step
- **CDN only** — Font Awesome 6 + Google Fonts

## Getting Started

```bash
git clone https://github.com/Taif6612/luxeglow-beauty-spa.git
cd luxeglow-beauty-spa
# Open index.html in browser or serve with: npx serve .
```
