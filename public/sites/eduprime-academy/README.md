# EduPrime Academy — Premium Coaching Center

A professional coaching center landing page built as a single dependency-free HTML file. EduPrime Academy prepares students for university admissions, board exams, competitive exams, and career success with expert faculty and proven results.

## Features

- **Hero Section** — Bold headline with animated stats and enrollment CTA
- **Courses & Programs** — Engineering, Medical, University Admission, IELTS, SAT, HSC, SSC, Job Exam preparation
- **Batch Schedule** — Upcoming batch timings, seat availability, and enrollment dates
- **Faculty Profiles** — Expert teacher bios with subject specializations
- **Results & Success Stories** — Student achievements and admission records
- **Testimonials** — Student and parent reviews with ratings
- **FAQ Accordion** — Common questions about courses, fees, and schedules
- **Contact & Enrollment Form** — Inquiry and registration form with toast notification
- **Fully Responsive** — Optimized for all screen sizes

## Tech Stack

- **HTML5** — Semantic, SEO-friendly structure with Open Graph meta tags
- **CSS3** — Custom properties, scroll-reveal animations, responsive grid
- **Vanilla JavaScript** — Zero dependencies, zero build step
- **CDN only** — Font Awesome 6 + Google Fonts

## Getting Started

```bash
git clone https://github.com/Taif6612/eduprime-coaching-center.git
cd eduprime-coaching-center
# Open index.html in browser or serve with: npx serve .
```
