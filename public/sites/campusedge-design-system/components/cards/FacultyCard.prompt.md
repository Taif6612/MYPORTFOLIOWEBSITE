Profile card for faculty / staff directories.

```jsx
<FacultyCard
  photo="https://images.unsplash.com/photo-..."
  name="Dr. Amara Okafor"
  role="Professor of Robotics"
  department="Engineering"
  qualification="Ph.D, MIT"
  research="Autonomous Systems & Control"
/>
```

Portrait desaturates slightly until hover. Use a 3–4 column grid for directory pages.
