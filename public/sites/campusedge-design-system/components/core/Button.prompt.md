Primary action button — use for every CTA across CampusEdge (Apply Now, Explore Programs, form submits).

```jsx
<Button variant="primary" size="lg" iconRight={<i className="fa-solid fa-arrow-right" />}>
  Apply Now
</Button>
```

Variants: `primary` (royal blue fill, default), `accent` (heritage gold — reserve for the single most important action), `outline` (secondary on light), `ghost` (inline/text-level), `glass` (over hero imagery). Sizes `sm | md | lg`. Use `as="a"` for links.
