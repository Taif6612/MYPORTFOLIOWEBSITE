Compact label for program categories, statuses, degree types, "Now enrolling" flags.

```jsx
<Badge tone="gold">Accredited</Badge>
<Badge tone="blue" soft={false}>Open House</Badge>
```

Tones: `blue` (default), `gold` (prestige), `navy`, `success`, `neutral`. `soft` (default) = tinted background; `soft={false}` = solid fill.
