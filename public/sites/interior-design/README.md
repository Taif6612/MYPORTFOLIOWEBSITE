# LuxeSpace Interiors — Luxury Interior Design Studio

A premium, fully responsive interior design studio landing page built as a single dependency-free HTML file. Designed for LuxeSpace Interiors, an award-winning luxury design and architecture studio based in Dhaka.

## Features

- **Hero Section** — Full-bleed imagery with animated headline and CTA
- **Portfolio Gallery** — Curated showcase of residential, commercial, hospitality, and retail projects
- **Services** — Detailed service offerings with elegant card layouts
- **About / Studio Story** — Brand narrative with team and philosophy sections
- **Process Walkthrough** — Step-by-step design process from concept to completion
- **Testimonials** — Client reviews with ratings and project references
- **FAQ Accordion** — Expandable common questions and answers
- **Contact Form** — Inquiry form with success state and toast notifications
- **Dark / Light Mode** — Elegant theme toggle with smooth transitions
- **Fully Responsive** — Graceful layout from widescreen down to mobile
- **Smooth Animations** — Scroll-reveal, hover lifts, and staggered entrances

## Tech Stack

- **HTML5** — Semantic, SEO-friendly structure with Open Graph meta tags
- **CSS3** — Custom properties, glassmorphism, scroll-reveal animations, responsive grid
- **Vanilla JavaScript** — Zero dependencies, zero build step
- **CDN only** — Font Awesome 6 icons + Google Fonts (Bodoni Moda / Jost)

## Getting Started

No installation or build tools required.

```bash
# Clone the repo
git clone https://github.com/Taif6612/interior-design.git
cd interior-design

# Open directly in browser, or serve locally:
npx serve .
```
