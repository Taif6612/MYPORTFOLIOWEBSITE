# SolarTech BD — Solar Energy Solutions

A professional, fully responsive solar energy company landing page built as a single dependency-free HTML file. Designed for SolarTech BD, offering residential, commercial, and industrial solar installations across Bangladesh.

## Features

- **Custom Animated Cursor** — Smooth cursor with blend-mode effects on desktop
- **Loading Screen** — Branded animated loader with progress bar
- **Dark / Light Mode** — Toggle with smooth CSS variable theming
- **Hero Section** — Bold headline with animated text reveals and CTA
- **Services Section** — Residential, commercial, and industrial solar solutions
- **Product Showcase** — Solar panel product cards with specs and pricing
- **Why Choose Us** — Trust indicators, stats counters, and key differentiators
- **How It Works** — Step-by-step installation process walkthrough
- **Project Gallery** — Completed installation case studies
- **Testimonials** — Customer reviews with star ratings
- **FAQ Accordion** — Expandable common questions and answers
- **Contact Form** — Inquiry form with success state and toast notifications
- **Fully Responsive** — Graceful layout from widescreen down to mobile

## Tech Stack

- **HTML5** — Semantic, SEO-friendly structure with Open Graph meta tags
- **CSS3** — Custom properties, glassmorphism, scroll-reveal animations, responsive grid
- **Vanilla JavaScript** — Zero dependencies, zero build step
- **CDN only** — Font Awesome 6 icons + Google Fonts (Inter / Syne)

## Getting Started

No installation or build tools required.

```bash
# Clone the repo
git clone https://github.com/Taif6612/solar-panel.git
cd solar-panel

# Open directly in browser, or serve locally:
npx serve .
```
