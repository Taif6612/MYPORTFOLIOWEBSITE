# Urban Bistro — Premium Restaurant & Café

A premium restaurant and café landing page built as a single dependency-free HTML file. Urban Bistro showcases handcrafted dishes, a signature menu, table reservations, and an elegant dark-themed dining experience.

## Features

- **Hero Section** — Full-bleed imagery with animated headline and reservation CTA
- **Menu Showcase** — Curated food and drinks menu with categories and pricing
- **Signature Dishes** — Featured items with descriptions and photography
- **Reservations** — Table booking form with date, time, and guest selection
- **About / Story** — Chef profiles and restaurant philosophy
- **Gallery** — Ambience and food photography grid
- **Testimonials** — Diner reviews with star ratings
- **Contact & Location** — Address, hours, and embedded map section
- **Dark Theme** — Elegant dark UI with warm accent tones
- **Fully Responsive** — Optimized for all screen sizes

## Tech Stack

- **HTML5** — Semantic, SEO-friendly structure with Open Graph meta tags
- **CSS3** — Custom properties, scroll-reveal animations, responsive grid
- **Vanilla JavaScript** — Zero dependencies, zero build step
- **CDN only** — Font Awesome 6 + Google Fonts (Anton / Cormorant Garamond / Jost)

## Getting Started

```bash
git clone https://github.com/Taif6612/urban-bistro-cafe.git
cd urban-bistro-cafe
# Open index.html in browser or serve with: npx serve .
```
